# 🤟 SignSpeak — Real-Time Hand Sign to Text & Speech

An end-to-end interactive computer vision and machine learning application that detects hand gestures via a webcam, converts them into text characters in real-time, builds them into words/sentences, and speaks the words aloud using Text-to-Speech (TTS).

Available in **two modes**:
- 🌐 **Web Dashboard** (recommended) — A beautiful browser-based UI with glassmorphism design, live hand skeleton overlay, and built-in speech synthesis.
- 🖥️ **Python / OpenCV** — Classic terminal + OpenCV camera window.

This pipeline utilizes **Google MediaPipe** for hand landmark extraction, **Scikit-Learn (Random Forest)** for sign classification, and either **Web Speech API** or **pyttsx3** for speech synthesis.

---

## ✨ Features

* **Real-Time Hand Tracking** — Extracts 21 3D hand landmarks at up to 30+ FPS using MediaPipe.
* **Custom Dataset Collector** — Quick script to record your own hand sign images via your webcam.
* **Random Forest Classification** — Robust and fast ML classifier with high accuracy.
* **Frame-Stability Detection** — Registers a letter only after the gesture is held stable for 15 frames, preventing accidental duplicates.
* **Beautiful Web Dashboard** — Premium dark-themed glassmorphism UI with animated progress bars, letter pop-in effects, and hand skeleton overlay.
* **Text-to-Speech** — Speaks your built sentence aloud (Web Speech API in browser, pyttsx3 in Python mode).
* **Keyboard Controls** — Full control over editing, clearing, and speaking your accumulated word/sentence.

## 🚀 Quick Start

### 1. Clone & Environment Setup

```bash
# Clone the repo
git clone <your-repo-url>
cd hand_sign_recognition

# Create and activate virtual environment (Windows)
python -m venv .venv
.venv\Scripts\Activate.ps1
# Or (CMD)
.venv\Scripts\activate.bat

# Install all dependencies
pip install -r requirements.txt
```

---

### 2. Train the Model (if starting fresh)

If you need to collect your own hand sign data and train a model:

#### Step 1 — Collect Data (`collect_data.py`)
Captures webcam images for each hand sign class (100 samples per class by default).
```bash
python collect_data.py
```
> Align your hand for a gesture and press **Q** to start capturing. Repeat for each letter.

#### Step 2 — Extract Landmarks (`create_dataset.py`)
Processes your captured images through MediaPipe to extract normalized coordinates.
```bash
python create_dataset.py
```
> Generates `data.pickle` containing structured landmarks and labels.

#### Step 3 — Train the ML Model (`train_model.py`)
Trains a Random Forest Classifier and prints accuracy.
```bash
python train_model.py
```
> Generates the trained model file `model.p`.

---

### 3. Run the App

You have two options to run the live recognition:

---

#### 🌐 Option A: Web Dashboard (Recommended)

The web dashboard runs **entirely in the browser** — no Python server needed after the initial model export.

**Step 1 — Export model to ONNX** (one-time setup):
```bash
# Install export dependencies
pip install skl2onnx onnxruntime

# Run the export script
python export_model_onnx.py
```
> This generates `web/model.onnx` and `web/labels.json`.

**Step 2 — Serve the web folder**:
```bash
# Start a local server (required for camera access)
python -m http.server 8000 --directory web
```

**Step 3 — Open in your browser**:
```
http://localhost:8000
```

> Click **"Start Camera"**, allow camera access, and start signing! 🎉

---

#### 🖥️ Option B: Python / OpenCV Mode

The classic mode using an OpenCV camera window:
```bash
python recognize_live.py
```

## ⌨️ Controls

These keyboard shortcuts work in **both** the web dashboard and the OpenCV window:

| Key | Action |
| :--- | :--- |
| **Hold a hand gesture** | Builds the sign when held stable for 15 frames (e.g. class `0` → **A**, class `1` → **B**, class `2` → **L**) |
| **`SPACEBAR`** | Adds a space between words |
| **`BACKSPACE`** | Deletes the last character |
| **`ENTER`** | Speaks the current sentence aloud |
| **`C`** | Clears the entire text |
| **`Q`** | Stops the camera / exits the app |

The web dashboard also has **clickable buttons** for Speak, Space, Delete, and Clear.
